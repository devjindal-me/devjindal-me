# Professional Frontend Developer Portfolio

A modern, responsive portfolio website for frontend developers to showcase their skills, projects, and experience.

![Portfolio Preview](https://via.placeholder.com/1200x630)

## Features

- **Responsive Design**: Fully responsive layout that looks great on all devices (mobile, tablet, desktop)
- **Dark Mode**: Toggle between light and dark themes
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Project Showcase**: Filterable project grid with detailed project information
- **Skills Visualization**: Visual representation of skills with animated progress bars
- **Testimonials**: Carousel slider to display client testimonials
- **Contact Form**: Functional contact form with validation
- **Smooth Scrolling**: Enhanced navigation with smooth scrolling to sections
- **Optimized Performance**: Fast loading times and smooth animations

## Technologies Used

- **HTML5**: Semantic HTML structure
- **CSS3**: Modern CSS with custom properties, flexbox, and grid
- **JavaScript**: Vanilla JavaScript for interactive elements
- **Font Awesome**: Icon library
- **Google Fonts**: Typography with Inter font family
- **CSS Animations**: Smooth transitions and effects

## Setup and Usage

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/frontend-portfolio.git
   ```

2. Open the project folder:
   ```
   cd frontend-portfolio
   ```

3. Open the `index.html` file in your browser, or use a local development server:
   ```
   npx serve
   ```

## Customization

### Personalization

1. Replace placeholder images with your own photos
2. Update project information in the HTML file
3. Modify the skills section to reflect your expertise
4. Update contact information and social media links

### Styling

- Customize the color scheme by modifying the CSS variables in the `:root` selector in `styles.css`
- Adjust fonts, spacing, and other visual elements as needed
